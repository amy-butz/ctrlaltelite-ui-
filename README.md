# Webapp Template
> A Web Application Template using [React](https://reactjs.org/) and [Marterial UI](https://material-ui.com/)
## Install
Using npm:
```sh
npm install
```
Using yarn:
```sh
yarn
```
## Build
Using npm:
```sh
npm run build
# To build UI only
npm run build-ui
# To build API only
npm run build-api
```
Using yarn:
```sh
yarn run build
# To build UI only
yarn run build-ui
# To build API only
yarn run build-api
```
## Run
### Development
Using npm:
```sh
npm run api-dev-server
npm run ui-dev-server
```
Using yarn:
```sh
yarn run api-dev-server
yarn run ui-dev-server
```
### Production
Using npm:
```sh
npm start
```
Using yarn:
```sh
yarn start
```
